# ArtVerse Server

This repository contains the **backend server** for ArtVerse, a creative artwork showcase platform. The server handles **artwork data, user authentication integration, and API endpoints** for the client-side React application.

## Features

- CRUD operations for artworks (Create, Read, Update, Delete).  
- User-specific data fetching based on Firebase authentication.  
- Search and filter artworks by category and title/artist.  
- Favorite management for users.  
- RESTful API endpoints for client consumption.  
- Handles image URLs and artwork metadata.  
- Lightweight and fast server using Node.js and Express.  

---

## Tech Stack

- **Runtime:** Node.js  
- **Framework:** Express.js  
- **Database:** MongoDB  
- **Authentication:** Firebase Auth (client-side)  
- **Other:** CORS, dotenv for environment variables
